# PI Barbearia - Django

Projeto base para sistema web de agendamento de barbearia com Django, PostgreSQL, GitHub e Render.

## Stack
- Python
- Django
- PostgreSQL
- Render
- GitHub

## Apps
- core
- clientes
- profissionais
- servicos
- agendamentos
- notificacoes

## Comandos iniciais
```bash
pip install -r requirements/dev.txt
python manage.py makemigrations
python manage.py migrate
python manage.py createsuperuser
python manage.py runserver
```
