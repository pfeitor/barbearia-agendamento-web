# Sistema Web de Agendamento para Barbearia

Este projeto consiste no desenvolvimento de um sistema web de agendamento para barbearias, com foco na organização de horários, redução de conflitos de agenda e diminuição de faltas sem aviso prévio.

O sistema permite que clientes realizem agendamentos online, selecionando o profissional e o horário disponível, além de receberem notificações automáticas via WhatsApp para confirmação do atendimento.

## Contexto Acadêmico

Este projeto está sendo desenvolvido como parte do **Projeto Integrador em Computação I**, do curso de Ciência de Dados / Engenharia de Computação / Ciência da Computação da **UNIVESP – Universidade Virtual do Estado de São Paulo**.

O trabalho segue a metodologia de Aprendizagem Baseada em Projetos e Design Thinking, sendo desenvolvido em colaboração com uma barbearia real da comunidade.

## Tecnologias Utilizadas

- **Linguagem:** Python  
- **Framework Web:** Django  
- **Banco de Dados:** PostgreSQL  
- **Notificações:** API WhatsApp (Evolution API)  
- **Versionamento:** Git e GitHub  
- **Hospedagem:** Render (plano gratuito)

## Objetivo do Sistema

Organizar o fluxo de agendamentos da barbearia, centralizando informações de clientes, profissionais e horários em um sistema único, acessível via navegador, reduzindo erros operacionais e melhorando a experiência de usuários e profissionais.
